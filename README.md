第一步：
https://github.com/CZJ-HEART/CZJ-OH
进入这个网址
进入之后点击绿色的按钮“code”
选https
然后里面有一个“Download zip”点击以后就可以下载源代码了



第二步：


将zip压缩文件放到你电脑的某一个固定位置，然后解压他
打开下面链接


https://nodejs.org/dist/v24.11.1/node-v24.11.1-x64.msi

下载node.js，不然你没办法安装依赖
（安装都看得懂，我视频里面会讲）



第三步：
对着你解压出来的文件夹，右键，用终端打开

输入命令：npm install

然后要等一会，等到你的文件名出来

然后在输入npm run dev

里面就有网址可以让你打开这个项目了。

其他细节可以看视频


说明一下如何更换树上的照片

根目录中的public\photos文件夹里（就是找到public文件夹，里面有个photos文件夹）

打开后就可以看到那些表情包

想更换的话直接将原本的表情包删掉，将你的jpg格式照片（什么照片都行）放进去，并且命名成（1.jpg）（2.jpg）这样的格式

最多放20张，当然会用ai或者懂代码的可以自己修改上限和逻辑
